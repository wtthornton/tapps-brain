Persistent, deterministic memory for AI coding agents — no raw text in the export by default. See recall mode, Hive sync, capacity, and trust signals for this brain. Load a snapshot from your project to get started, or use Load snapshot below.

Refresh data (CLI and files)

From a project with the CLI: run tapps-brain visual export -o brain-visual.json (add --skip-diagnostics for speed; use --privacy strict or --privacy local for redaction or tag/group detail).

Serving this folder over HTTP loads brain-visual.json when it sits next to index.html. Opening the HTML as file:// uses Load snapshot instead. Also loads scorecard-derive.js and brain-visual-help.js.

Privacy

Memory content is never exported. Aggregates only: counts, histograms, and schema versions. Tag and group names appear only with --privacy local (single-user, local-trust mode).

  • 🚫

    Excluded

    Memory content, entry keys, raw queries, and session IDs are never included in any export tier.

  • 📦

    Aggregated

    Counts, tier histograms, access buckets, schema versions, and health scores — no per-entry data.

  • 🏠

    Local-only

    Tag and group names appear only in --privacy local mode (single-user, local-trust stores).
