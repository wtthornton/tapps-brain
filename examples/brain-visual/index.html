About this dashboard

This page polls the live /snapshot endpoint on the running tapps-brain HTTP adapter every 30 seconds (configurable via the interval selector). There is no file-load fallback — if the endpoint is unreachable, start the adapter.

? buttons open a side panel with definitions, math, and code references aligned to tapps-brain (not generic marketing text). KPI strips, section headers, bento tiles, and each scorecard row can link to an article; some tiles expose two topics (e.g. Hive hub vs agent scope).
